Creates a new empty CC_Array based on the specified CC_ArrayConf struct and returns a status code.

The CC_Array is allocated using the allocators specified in the CC_ArrayConf struct. The allocation may fail if underlying allocator fails. It may also fail if the values of exp_factor and capacity in the CC_ArrayConf do not meet the following condition: exp_factor < (CC_MAX_ELEMENTS / capacity).

Parameters
[in]confarray configuration structure
[out]outpointer to where the newly created CC_Array is to be stored
Returns
CC_OK if the creation was successful, CC_ERR_INVALID_CAPACITY if the above mentioned condition is not met, or CC_ERR_ALLOC if the memory allocation for the new CC_Array structure failed.
